# Mercedes-F1
🏎️ Mercedes-AMG Petronas F1 Team Fan Website
A modern, responsive fan website dedicated to the Mercedes-AMG Petronas Formula One Team. This project showcases the team's 2025 lineup, technical specifications of the W16 car, and their historic achievements in motorsport.

✨ Features
Responsive Design: Fully optimized for desktops, tablets, and mobile devices.

Dynamic UI: Features smooth scroll animations, glowing text effects, and interactive elements.

Comprehensive Data: Includes the full 24-race 2025 F1 calendar, driver standings, and team records.

Modern Tech: Built with a focus on clean CSS architecture and performance.

🛠️ Tech Stack
HTML5: Semantic structure.

CSS3: Custom properties (variables), Flexbox, CSS Grid, and Keyframe animations.

JavaScript: Mobile menu logic and custom "Scroll-to-Top" functionality.

Assets:  for icons and  (Orbitron & Roboto).

📂 Sections Included
Drivers: Profiles for George Russell and Kimi Antonelli.

The Car: In-depth look at the W16 Silver Arrow.

Records: Historical data and pit stop milestones.

Sustainability: Overview of the "Net Zero 2030" initiative.

🚀 How to Run
